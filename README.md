## kilo-clone

a text editor written in c, following the [Build Your Own Text Editor](https://viewsourcecode.org/snaptoken/kilo/index.html) instruction booklet. based off [antirez's kilo](http://github.com/antirez/kilo)

### usage

```
make
./kc filename
```
